# Веб-приложение для создания опросов

По функционалу похож на google forms

>Создайте файл /.env с переменными:

    POSTGRES_DB_PASSWORD=
    POSTGRES_DB_NAME=
    POSTGRES_DB_USER=
    DJANGO_SECRET_KEY=
 
Для запуска в докере:

    docker-compose build
    docker-compose up

Доступен по адресу localhost:8080
